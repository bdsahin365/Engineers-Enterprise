# Engineers Enterprise

This repository contains the source code for the Engineers Enterprise application, structured as a monorepo.

## Project Structure

- **frontend/**: The React frontend application (Vite).
- **backend/**: The Strapi backend application.

## Getting Started

### Backend

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run develop
   ```

### Frontend

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
